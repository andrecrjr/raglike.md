# Setup Guide

To set up this system, you need [Bun](https://bun.sh) installed.

## Quick Start

1. **Install dependencies**:
   ```bash
   bun install
   ```

2. **Run the server**:
   ```bash
   # Default MCP mode (Stdio)
   bun start

   # HTTP API mode
   bun start --api
   ```
